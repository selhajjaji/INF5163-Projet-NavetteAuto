\section{Responsabilité, documentation et propriété intellectuelle}

Nous discutons ici des enjeux de responsabilité en cas d’accident :
fabricant, développeurs, municipalité, opérateurs.

La section inclut également :
\begin{itemize}
    \item un plan de documentation (journalisation, traçabilité des décisions) ;
    \item les enjeux de souveraineté des données collectées dans l’espace public ;
    \item les conflits potentiels entre innovation commerciale et protection citoyenne.
\end{itemize}
