\section{Rappel du schéma méthodologique}

La méthodologie hypothétique utilisée pour NavetteAuto comprend :

\begin{itemize}
    \item un module de perception basé sur la fusion LiDAR-caméra ;
    \item un modèle de détection d’obstacles reposant sur l’apprentissage profond ;
    \item un algorithme de décision utilisant l’apprentissage par renforcement ;
    \item un simulateur urbain pour valider les trajectoires en environnement complexe.
\end{itemize}

Cette section résume le cadre technique sans entrer dans les détails d’implémentation.
