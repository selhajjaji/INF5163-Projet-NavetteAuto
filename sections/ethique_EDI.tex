\section{Analyse d’intégrité, d’éthique et d’ÉDI}

Cette section analyse les dilemmes moraux associés aux navettes autonomes :
scénarios d’accident inévitable, choix programmés (utilitariste, déontologique),
gestion éthique de la surveillance, collecte et anonymisation des données.

L’analyse ÉDI inclut :
\begin{itemize}
    \item évaluation de l’accessibilité pour les personnes à mobilité réduite ;
    \item biais du modèle de perception selon les profils de piétons ;
    \item robustesse dans les conditions variées (neige, pluie, faible luminosité).
\end{itemize}
