\section{Plan de valorisation et diffusion}

Deux axes sont présentés :

\textbf{Valorisation :}
\begin{itemize}
    \item partenariats public-privé ;
    \item offres technologiques aux manufacturiers ;
    \item création d’un laboratoire vivant (« living lab ») dans une ville partenaire.
\end{itemize}

\textbf{Diffusion :}
\begin{itemize}
    \item communication avec les autorités réglementaires (Transport Canada) ;
    \item stratégie pour obtenir l’acceptabilité sociale ;
    \item preuves de sécurité et transparence technique.
\end{itemize}
